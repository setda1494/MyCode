Date 객체 생성과 초기화

    Date 객체의 주요 method

  1. getFullYear( ): 연도를 구하는 메소드
  2. getMonth( ) : 월을 구하는 메소드(배열로저장되어 시작인덱스 0)
  3. getDate( ) : 일자를 구하는 메소드
  4. getDay( ) : 요일을 구하는 메소드
  5. getHour( ) : 시간을 구하느 메소드
  6. getMinutes( ): 분을 구하는 메소드
  7. getSeconds( ) : 초를 구하는 메소드
    8. 배경색변경하기 / 날짜구하기
  8. setYear( ): 연도를 정하는 메소드
  9. setMonth( ) : 월을 정하는 메소드(배열로저장되어 시작인덱스 0)
  10. setDate( ) : 일자를 정하는 메소드
  11. setDay( ) : 요일을 정하는 메소드
  12. setHour( ) : 시간을 정하느 메소드
  13. setMinutes( ): 분을 정하는 메소드
  14. setSeconds( ) : 초를 정하는 메소드

  15. setInterval( ) : 일정간격으로 호출하는 메소드
  16. clearTimeout( ) : 초를 정하는 메소드
